# نمایش دموی پنل در GitHub Pages

این نسخه فقط برای نمایش ظاهر پنل به مشتری آماده شده است. اطلاعات، آمار، سفارش‌ها و دکمه ذخیره، نمایشی هستند و به پایگاه‌داده وصل نیستند.

## انتشار

1. همه فایل‌های پروژه را در شاخه `main` مخزن GitHub قرار دهید.
2. وارد `Settings` مخزن شوید.
3. از منوی سمت چپ `Pages` را باز کنید.
4. در بخش `Build and deployment`، گزینه `Source` را روی `GitHub Actions` بگذارید.
5. وارد تب `Actions` شوید و منتظر سبزشدن اجرای `Deploy static admin demo` بمانید.

پس از پایان انتشار، نشانی پنل به این شکل خواهد بود:

`https://USERNAME.github.io/REPOSITORY/`

اگر مخزن خصوصی باشد یا GitHub Pages برای حساب شما فعال نباشد، GitHub همان‌جا پیام مربوط را نمایش می‌دهد.
