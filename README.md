# پنل مدیریت آجر جهان‌نما

پنل فارسی و راست‌چین برای مدیریت محتوای سایت، محصولات، سفارش‌ها، شماره مشتریان و گزارش بازدید.

## امکانات

- ویرایش متن صفحه اصلی، شماره‌ها و آدرس با پیش‌نمایش زنده
- افزودن، حذف، فعال/غیرفعال‌کردن و ویرایش محصولات
- جستجو در سفارش‌ها و دریافت خروجی CSV
- داشبورد ساده آمار و محل اتصال Google Analytics
- ذخیره دائمی محتوا و محصولات در Cloudflare D1
- طراحی واکنش‌گرا برای موبایل و دسکتاپ

## اجرای پروژه

نیازمندی: Node.js نسخه 22.13 یا جدیدتر

```bash
npm install
npm run dev
```

سپس آدرسی که ترمینال نمایش می‌دهد را در مرورگر باز کنید.

## آماده‌سازی نسخه نهایی

```bash
npm run build
```

## نکته اتصال به سایت اصلی

فرم سفارش سایت فروشگاهی باید اطلاعات مشتری را در جدول `orders` همین پایگاه داده ثبت کند. برای نمایش آمار واقعی نیز شناسه Google Analytics مجموعه در بخش تنظیمات وارد و اتصال API گزارش‌گیری در محیط سرور فعال شود. داده‌های آماری فعلی صرفاً نمونه رابط کاربری هستند.

پنل مدیریت را عمومی نکنید؛ پیش از استفاده واقعی، دسترسی آن را فقط به مدیران مجاز محدود کنید.
